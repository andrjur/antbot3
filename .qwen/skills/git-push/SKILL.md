# Git Push Skill

## Автоматический пуш изменений на GitHub

### Команда
```
git-push
```

### Что делает:
1. `git status` - проверка состояния
2. `git add .` - добавление всех изменений
3. `git commit -m "<сообщение>"` - коммит
4. `git push antbot4 main --force` - пуш в remote `antbot4`
5. `git status` - подтверждение что пуш успешен

### Использование в чате:
Пользователь пишет: "запушь изменения" или "git push"
→ Скрипт автоматически делает все шаги выше

### Важно:
- Remote должен называться `antbot4`
- Ветка: `main`
- Force push включён для гарантии
